# Security Policy

## Supported version

Security fixes are applied to the latest released V0 version.

## Reporting a vulnerability

Do not open a public issue for a suspected vulnerability. Use [GitHub's private security advisory form](https://github.com/remyjallan/impresairio/security/advisories/new) with reproduction steps, impact and a suggested fix if available. You will receive an acknowledgement within seven days.

Avoid including credentials, API keys, private run state or personal documentation paths in the report.
